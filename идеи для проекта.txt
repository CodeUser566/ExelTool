1) Конвертер CSV ↔ Excel (выполнено)
Описание:
Программа, которая преобразует CSV-файлы в формат Excel (.xlsx).
Функционал:

Чтение CSV и запись данных в Excel-файл.



2) Генератор календаря
Описание:
Создает Excel-файл с календарем на указанный месяц или год.
Функционал:

Автоматическое заполнение дат, дней недели.
Выделение выходных (суббота/воскресенье) цветом.


3) Трекер расходов
Описание:
Программа для учета личных финансов. Данные сохраняются в Excel-файл.
Функционал:

Ввод расходов/доходов через консоль или GUI (если добавить Qt).
Автоматическая категоризация трат (еда, транспорт и т.д.).
Генерация сводки: суммарные расходы за месяц, графики (если подключить внешние библиотеки для визуализации).


4) Генератор тестовых данных
Описание:
Заполняет Excel-файл случайными данными для тестирования (например, имена, даты, числа).
Функционал:

Выбор типа данных (строки, числа, даты).
Настройка количества строк/столбцов.
Добавление "шапки" таблицы с стилями.


5) Шахматная доска в Excel
Описание:
Создает шахматную доску в Excel, где каждая ячейка соответствует клетке доски.
Функционал:

Раскраска ячеек в шахматном порядке.
Добавление букв (A-H) и цифр (1-8) по краям.
Возможность отмечать ходы (например, ввод с клавиатуры).


6) Парсер логов в Excel
Описание:
Читает текстовый лог-файл (например, сервера) и сохраняет структурированные данные в Excel.
Функционал:

Фильтрация записей по уровню (ERROR, WARNING, INFO).
Подсчет количества событий каждого типа.
Выделение ошибок красным цветом.


7) Генератор расписания
Описание:
Создает Excel-расписание занятий или рабочих задач.
Функционал:

Ввод данных через консоль (предмет, время, аудитория).
Автоматическое распределение по дням недели.
Проверка на пересечение времени.


8) Сканер акций (мини-версия)
Описание:
Программа, которая запрашивает данные акций через API (например, Yahoo Finance) и сохраняет их в Excel.
Функционал:

Запрос цены акций на текущий момент.
Сохранение истории в отдельный лист.
Автоматический расчет изменения цены за период.


9) Генератор сертификатов
Описание:
Создает Excel-файл с сертификатами участников (например, для онлайн-курса).
Функционал:

Ввод списка имен.
Автоматическое заполнение шаблона сертификата (каждый на отдельном листе).
Добавление даты выдачи и подписи.


10) Симулятор бюджетирования для игры
Описание:
Генератор Excel-таблицы для управления ресурсами в настольной или компьютерной игре (например, как в SimCity).
Функционал:

Ввод доходов/расходов "виртуального города".
Автоматический расчет баланса (например, налоги, затраты на инфраструктуру).
Визуализация статистики через условное форматирование (например, красный цвет при убытках).


11) Планировщик учебного расписания
Описание:
Программа, которая распределяет учебные курсы по семестрам, избегая пересечений.
Функционал:

Ввод списка курсов с часами и требованиями.
Автоматическое распределение по семестрам.
Визуализация расписания в виде таблицы с цветовой маркировкой.


12) Генератор персональных QR-кодов
Описание:
Программа сохраняет в Excel список персональных QR-кодов (например, для визиток), используя внешние библиотеки для генерации QR.
Функционал:

Ввод данных (имя, email, телефон) через консоль.
Генерация QR-кода для каждой записи (через библиотеку вроде libqrencode).
Вставка изображений QR-кодов в ячейки (если XLNT поддерживает вставку изображений).


13) Анализатор текста
Описание:
Программа анализирует текст (например, из файла) и сохраняет в Excel статистику: частоту слов, длину предложений, использование ключевых слов.
Функционал:

Подсчет слов, символов, стоп-слов.
Построение гистограмм частотности слов (через Excel-формулы).
Выделение редко используемых терминов.


14) Генератор случайных персонажей для RPG
Описание:
Создает таблицу с параметрами персонажей для настольных игр (D&D, Pathfinder).
Функционал:

Случайная генерация характеристик (сила, ловкость, интеллект).
Добавление расы, класса и инвентаря.
Экспорт в Excel с цветовым кодированием параметров.
